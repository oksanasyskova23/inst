# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсива необходимо обрамит его звездочками(*). Например,*вот так*.

Что бы выделить текст полужирным, необходимо обрамить его двойными звездочками(**).
Например,**вот так**.
